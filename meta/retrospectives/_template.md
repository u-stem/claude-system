## failure-log.jsonl の繰り返しパターン

集中しているならその領域の hook 追加 / skill 化を検討。

```bash
# 集計コマンド例
jq -r '.category' ~/.claude/projects/*/failure-log.jsonl 2>/dev/null | sort | uniq -c | sort -rn | head
```
